您可以為使用者指派以下角色，以便他們存取BlueXP中的Google Cloud NetApp Volumes 。

Google Cloud NetApp Volumes使用以下角色：

* * Google Cloud NetApp Volumes管理員*：在BlueXP中發現並管理Google Cloud NetApp Volumes 。
