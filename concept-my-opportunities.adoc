= 我的商機
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
在畫版上、「*我的商機*」索引標籤提供集中位置、可探索可新增至藍圖XP的現有資源、以便在混合式多雲端上提供一致的資料服務與作業。

目前、My Opportunity可讓您在ONTAP AWS帳戶中探索現有的FSXfor Solidffile系統。
