= 尋找連接器的系統ID
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
為協助您開始使用、NetApp代表可能會要求您提供Connector的系統ID。此 ID 通常用於授權和疑難排解。

.步驟
. 在BlueXP主控台右上角、按一下「說明」圖示。
. 按一下*支援> Connector*。
+
系統ID會顯示在頂端。
