= BlueXP backup and recovery角色
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以為使用者指派以下角色，以便他們存取BlueXP中的備份和復原服務。備份和復原角色讓您可以靈活地為使用者指派特定於組織內需要完成的任務的角色。如何分配角色取決於您自己的業務和儲存管理實踐。

備份和還原使用以下角色：

* *備份和還原超級管理員*：執行任何操作。
* *備份和還原管理*：執行本機快照備份、複製到二級儲存以及備份到物件儲存。
* *備份和還原還原管理*：還原工作負載。
* *備份和還原克隆管理*：克隆應用程式和資料。
* *備份和還原檢視器*：查看備份和還原資訊。
