= 內部連接器的連接埠
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
當 Connector 手動安裝在內部部署 Linux 主機上時、 Connector 會使用 _inbound 連接埠。您可能需要參考這些連接埠以進行規劃。

這些輸入規則適用於所有 BlueXP 部署模式。

[cols="10,10,80"]
|===
| 傳輸協定 | 連接埠 | 目的 


| HTTP | 80 | 提供從用戶端 Web 瀏覽器到本機使用者介面的 HTTP 存取 


| HTTPS | 443.. | 提供 HTTPS 存取、從用戶端網頁瀏覽器存取本機使用者介面 
|===
