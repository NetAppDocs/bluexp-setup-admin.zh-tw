= BlueXP disaster recovery角色
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
您可以為使用者指派以下角色，以便他們存取BlueXP中的備份和復原服務。備份和復原角色讓您可以靈活地為使用者指派特定於組織內需要完成的任務的角色。如何分配角色取決於您自己的業務和儲存管理實踐。

災難復原使用以下角色：

* *災難復原管理員*：執行任何動作。
* *災難復原故障轉移管理*：執行故障轉移和遷移。
* *災難復原應用程式管理員*：建立複製計劃。修改複製計劃。啟動測試故障轉移。
* *災難復原檢視器*：僅查看資訊。
